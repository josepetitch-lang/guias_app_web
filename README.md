Este repositorio contiene una aplicación web desarrollada en **Python** utilizando el framework **Flask**, diseñada como un repositorio práctico para almacenar y visualizar fórmulas científicas.

PD: PARECE LENGUAJE LINKEDIN JSJSJSJSJS
## Estructura del Proyecto

* **ejercicio/**: Tiene las plantillas (`render.html`), los formularios y la lógica para crear subgrupos por asignatura.
* **practica/**: La carpeta donde tiro código para hacer pruebas y testear cosas antes de moverlas.

## Tecnologías Utilizadas

* **Python 3.13**
* **Flask** (Manejo de rutas y servidor local)
* **Jinja2** (Renderizado de plantillas HTML)
* **HTML**
